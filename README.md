# Light of Day — Tarot Reading

A beautiful, mystical online tarot card reading experience built with pure HTML, CSS, and JavaScript. No frameworks, no dependencies, no API keys required.

## Features

- **Single Card Draw** — Draw one card with shuffle animation, 3D card flip, sparkle effects
- **Three Card Spread** — Past · Present · Future reading with staggered reveals
- **Card Library** — Browse all 78 tarot cards with detailed upright & reversed meanings
- **Responsive Design** — Works on desktop, tablet, and mobile
- **Animated Star Background** — Procedurally generated twinkling stars
- **Original Card Designs** — CSS-rendered cards with no copyrighted artwork
- **30% Reversed Chance** — Authentic reversed card mechanic

## Card Data

Full 78-card traditional tarot deck:
- **22 Major Arcana** (The Fool through The World)
- **14 Wands** (Fire — passion, creativity, willpower)
- **14 Cups** (Water — emotions, love, intuition)
- **14 Swords** (Air — intellect, truth, conflict)
- **14 Pentacles** (Earth — material, wealth, health)

## Tech Stack

- Pure HTML5 / CSS3 / ES6 JavaScript
- Google Fonts (Cinzel + Cormorant Garamond)
- No build step, no dependencies
- GitHub Pages ready

## Getting Started

Just open `index.html` in a browser, or serve with any static file server:

```bash
# Python
python -m http.server 8000

# Node
npx serve .
```

## Deployment (GitHub Pages)

1. Push to `main` branch
2. Go to repo Settings → Pages
3. Set source to `main` branch, root folder
4. Site will be live at `https://<username>.github.io/light-of-day/`
